# Sudoku - Nadia Naba
## Description 
This project allows you to generate a finished Sudoku board, or play a game of Sudoku!

## Instructions

## Summary
Each time the program is run, a Board is generated by creating a 2D array, randomly flooding the 3x3 squares on the main diagonal, and recursively filing the rest of the cells in order to generate a valid Sudoku board.
